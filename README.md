📌 What is MERN Stack?

The MERN stack consists of:

MongoDB – NoSQL database

Express.js – Backend framework for Node

React – Frontend library

Node.js – JavaScript runtime

All technologies use JavaScript, making it a full-stack JavaScript solution.
